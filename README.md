## NEERJA-A research Project

**ocean-NEERJA** is a comprehensive marine-environment analysis and monitoring system built to study salinity patterns, fish migration, marine biodiversity, and ocean-health indicators.  
This repository contains datasets, models, server code, and front-end modules used during development.
